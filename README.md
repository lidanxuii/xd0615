## 0615课堂笔记

1. 关于git
-  cd 打开一个文件
-  cd..\ 退到上一文件夹
-  cat 查看一个文件的内容
-  vim 编辑某文件内容
-  git clone URL 克隆链接内容到本地
-  git add 把我的更新加到我的仓库
-  git commit -m "" 对修改内容添加注释
-  git push 上传到github
